## המדריכים הזמינים

### [הפקת סט כושר רציף באייפד: המדריך המעשי והמינימליסטי](cubasis-fitness-guide.md)

מדריך מקיף בעברית המלמד כיצד ליצור סט מוזיקה רציף של 50 דקות לשיעורי כושר באמצעות Cubasis 3 על אייפד.

**מה תלמד:**
- ארגון שירים והכנה לפני העבודה
- יצירת פרויקט והגדרת BPM
- ייבוא שירים ויישור לגריד
- יצירת מעברים חלקים בין שירים
- ווליום ומיקס בסיסי
- ייצוא הסט כקובץ MP3

**מתאים במיוחד למדריכי כושר ומפיקים מתחילים עם ADHD** - המדריך כתוב בצורה ברורה, מינימליסטית ופרקטית.
